# toyboard
一款使用8x8LED矩阵的Arduino Esplora兼容开发板，基于该开发板，我们可以开发一些复古的像素风游戏。
##目前可用的游戏
- 贪吃蛇(https://github.com/uname/toyboard/tree/master/snake)

观看贪食蛇效果视频(http://v.youku.com/v_show/id_XMTI3NzI1MTkyNA==.html)
